# System Vulnerability Checklist

## Description
A beginner-friendly guide for identifying basic security vulnerabilities across five key areas of a system. Provides structured checklists and associated risks to help individuals and organizations assess their security posture without requiring advanced cybersecurity expertise.

## How to Run
Open `network_vulnerability_checklist.pdf` to access the complete checklist presentation. The checklist covers:

- **Weak Password Check** — password strength, default credential changes, and MFA
- **Software Update Check** — OS, antivirus, application patching, and auto-updates
- **User Practices Check** — phishing awareness, safe browsing, and security training
- **Network Security Check** — firewall, Wi-Fi security, port management, and VPN
- **Data Protection Check** — backups, encryption, access permissions, and recovery testing
